# **App Name**: Joysriram's Digital Canvas

## Core Features:

- Multilingual Content Toggle: Instantly switch between Bengali and English content using a language toggle.
- Dynamic Content Loading: Load content dynamically using JavaScript based on the selected language, without page reloads.
- Hero Section with Dual-Language Support: Display a headline and sub-headline in either Bengali or English, with appropriate translations.
- Stats Showcase: Showcase key stats (experience, projects, dedication, support) relevant to skills, localized as appropriate.
- Services Presentation: Display the range of provided services, such as web development and content writing, with icon cards. All service descriptions must change language with the selected locale.

## Style Guidelines:

- Primary color: Gold/Yellow (#FFD700) for highlights, buttons and active states.
- Background color: Deep Black (#1a1a1a) to provide a professional dark-themed aesthetic.
- Accent color: A slightly lighter shade of Gray (#333333) for secondary text and borders.
- Headline font: 'Poppins' (sans-serif) for English headings and key text, ensuring modern clarity. Note: currently only Google Fonts are supported.
- Body font: 'Hind Siliguri' (sans-serif) for Bengali and supporting text, ensuring proper rendering and readability. Note: currently only Google Fonts are supported.
- Responsive layout to ensure seamless viewing experience across devices.
- Smooth transitions and subtle animations for improved user experience.
- Clean, minimalist icons for services and other visual elements, maintaining a consistent style.
